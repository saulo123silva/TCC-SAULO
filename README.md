Instale as bibliotecas necessárias (caso ainda não tenha instalado). Abra o seu terminal ou prompt de comando e digite:

Bash
pip install PyQt5 matplotlib numpy
Salve o código em um arquivo de texto com a extensão .py (por exemplo, tcc_saulo.py).

Rode o arquivo pelo terminal ou pela sua IDE favorita (como VS Code, PyCharm ou Spyder):

Bash
python tcc_saulo.py
